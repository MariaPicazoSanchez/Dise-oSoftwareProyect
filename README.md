#Factory Method:
```
Crea la clase Game con los atributos maze, bichos e hilos.
Crea dentro de la clase Game create_wall que devuelve un Wall, un cretae_ttoom que devulva un objeto Room con el id introducido, un create_maze que devulve un objeto de Maze.
Crea dentro de la clase Game create_door que dado dos lados creamos un objeto Door con estos dos lados.
```
